# Typing Speed Test Game

A console-based typing speed testing game written in C that measures a user's typing performance through Words Per Minute (WPM) and Accuracy metrics.

## Features

### Two Game Modes

#### Test Mode

The primary evaluation mode designed to assess typing performance.

* Measures Words Per Minute (WPM)
* Measures typing accuracy
* Supports multiple users
* Displays rankings based on performance
* Uses fixed-length paragraphs for fair evaluation

#### Practice Mode

A training mode designed to improve keyboard familiarity and typing speed.

* Provides short sentences for practice
* Focuses on developing typing habits and speed
* Less emphasis on accuracy
* Suitable for beginners and warm-up sessions

### Difficulty Levels

Both game modes support three difficulty levels:

* Easy
* Medium
* Hard

Each difficulty level contains a different set of typing scripts appropriate for that level.

### User Ranking System

* Accepts usernames from players
* Stores player performance
* Displays rankings based on WPM and accuracy scores

## Technologies Used

* C Programming Language
* GCC Compiler

## Compilation

Compile the program using GCC:

```bash
gcc SDFProject.c -o SDFProject
```

## Running the Program

```bash
./SDFProject
```
